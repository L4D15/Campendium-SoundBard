# Changelog

All notable changes to SoundBard are documented here. This project follows [Semantic Versioning](https://semver.org/).

## [1.1.0] - 2026-06-21

### Bug Fixes
- Fixed an issue where closing SoundBard would sometimes cause the panel to reappear after clicking on a different module.

### New
- You can now hear reverb and EQ changes live as you adjust their values.
- Added buttons to reset EQ bands back to their default settings.
- Updated EQ band frequencies to allow for greater flexibility.

## [1.0.0] - 2026-06-21

### New
- Initial release.
